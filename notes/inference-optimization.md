# Inference Optimization Basics

## Focus points

- Latency: how long one request takes.
- Throughput: how many requests can be served in a time window.
- Batching can improve throughput.
- Quantization can reduce model size and memory usage.
- GPU inference can improve performance for large models.
- Monitoring is required to detect slow inference, errors, and resource bottlenecks.
