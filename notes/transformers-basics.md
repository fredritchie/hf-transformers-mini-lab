# Transformers Basics

## Focus points

- Transformers process tokens, not raw text.
- Tokenizers convert text into `input_ids` and `attention_mask`.
- Pretrained models can be used through pipelines or lower-level model/tokenizer APIs.
- Different tasks use different model heads.
- Inference means using a trained model to generate predictions.
