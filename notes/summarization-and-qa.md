# Summarization and Question Answering

## Summarization

Useful for:

- document summaries
- report generation
- long text compression
- AI assistant workflows

## Question answering

Useful for:

- document Q&A
- PDF chatbot
- retrieval augmented generation
- structured extraction from context

## Key idea

Question answering needs context. RAG systems retrieve context first, then use the LLM to answer.
