# Hugging Face Datasets

## Focus points

- Use datasets to load and process structured data.
- Useful methods: `select`, `filter`, `map`, `shuffle`, `train_test_split`.
- Custom datasets can be loaded from CSV, JSON, and text files.
- Streaming helps when datasets are too large to fit into memory.

## AI infrastructure connection

Datasets are useful for:

- RAG ingestion
- evaluation datasets
- model training pipelines
- reproducible preprocessing
- batch embedding generation
